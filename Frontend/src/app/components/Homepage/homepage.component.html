This site is an interactive platform designed to track artists
and their trends over time.
It also serves as a platform for me to test things on.

Technologies Used

  • Front-End: Angular, HTML5, CSS3
  • Real-Time Communication: SignalR
  • Back-End: C# ASP.NET Core microservices
  • Messaging: RabbitMQ
  • Database: MySQL
  • Package Management: npm & NuGet (via dotnet CLI)
  • Version Control: Git & GitHub
  • Deployment: Docker and a homeserver
